# 前端设计规范(Design Brief)

> 参考 **JumpServer** 的配色与设计风格(默认主题:经典绿 Classic Green)。
> 本规范是所有前端页面实现的**基点**。任何页面开工前先对齐本文件。
> **硬约束:整体设计不许让人一眼看穿是 AI 产物**(见本文末「反 AI 感清单」)。

## 1. 设计定位

企业级运维控制台,气质对标 JumpServer:**克制、专业、信息密度高、绿色点缀而非绿色泛滥**。不是营销落地页,不做花哨渐变、不做大圆角卡片堆叠、不做居中英雄区。

## 2. 配色(参考 JumpServer 经典绿,精确值以实际组件库为准)

| 角色 | 色值 | 用途 |
|------|------|------|
| 主色 Primary | `#1AB394`(经典绿,INSPINIA 系) | 主按钮、选中态、关键强调、链接 hover |
| 主色深 | `#18A689` | 按钮 hover/active |
| 侧边栏底 | `#2F4050`(深墨蓝灰) | 左侧导航背景 |
| 侧边栏文字 | `#A7B1C2` | 导航默认文字 |
| 侧边栏选中 | `#293846` 底 + 左侧 `#1AB394` 3px 高亮条 | 当前菜单 |
| 顶栏 | `#FFFFFF` + 底部 1px `#E7EAEC` 分隔 | 顶部工具条 |
| 内容区底 | `#F3F3F4`(极浅灰) | 页面背景,衬托白色卡片 |
| 卡片/面板 | `#FFFFFF` + 1px `#E7EAEC` 边框 | 内容容器 |
| 正文文字 | `#676A6C` | 主要文字(注意:不是纯黑,偏灰) |
| 标题文字 | `#333333` | 标题 |
| 成功/运行 | `#1AB394` | running / success |
| 警告 | `#F8AC59` | warning / 部署中 |
| 危险/异常 | `#ED5565` | error / failed / critical |
| 信息 | `#1C84C6` | info / 链接 |

**关键**:绿色是**点缀色**,大面积用中性灰白 + 深墨蓝灰侧边栏。全屏铺绿或彩色渐变是错的。

## 3. 版式与间距

- 字体:系统无衬线栈(`-apple-system, "Segoe UI", "Microsoft YaHei", sans-serif`),**不引入 Google Fonts** 那种一眼 AI 味的字体。
- 正文 13px,标题 14~16px(JumpServer 偏小字号、高密度)。
- 圆角 **2~3px**(小而克制),**不用 12px+ 大圆角**。
- 阴影极淡或无(`0 1px 2px rgba(0,0,0,.05)`),**不用发光/彩色阴影**。
- 栅格紧凑,表格是一等公民(运维数据密集)。

## 4. 布局骨架

```
┌───────────┬──────────────────────────────────────┐
│           │  顶栏:面包屑 · 环境切换 · 用户菜单      │
│  深色侧栏  ├──────────────────────────────────────┤
│  Logo     │                                       │
│  服务器    │   内容区(浅灰底 + 白色卡片/表格)       │
│  服务      │                                       │
│  部署      │                                       │
│  配置      │                                       │
│  监控      │                                       │
└───────────┴──────────────────────────────────────┘
```

- 左侧固定深色导航(可收起为图标条),二级菜单展开。
- 顶栏放面包屑 + 环境(dev/staging/prod)切换 + 用户/告警入口。
- 内容区以表格、表单、详情三段式为主。

## 5. 组件基调(Ant Design 定制)

用 Ant Design 但**必须改 token**贴合上表,否则就是一眼 antd 默认蓝的 AI 感。关键 token:`colorPrimary=#1AB394`、`borderRadius=2`、`colorBgLayout=#F3F3F4`、`fontSize=13`。侧边栏用深色 Menu 主题。

## 6. 反 AI 感清单(约束3,逐条自检)

做完任一页面,对照自检,命中即改:

- [ ] **没有**居中大标题 + 副标题 + 一个大 CTA 的"落地页三件套"。
- [ ] **没有**紫蓝渐变、玻璃拟态、霓虹发光、超大圆角卡片。
- [ ] **没有**每个模块都配一句正确的废话副标题(AI 最爱)。
- [ ] 主色是点缀而非铺满;中性灰白占主导。
- [ ] 表格/表单是主体,信息密度高,不是留白撑满一屏放三个卡片。
- [ ] 文案用运维行话、简洁祈使句,不用"轻松管理您的一切"这类营销腔。
- [ ] 图标克制统一(一套线性图标),不混用 emoji。
- [ ] 有真实的边界态:空状态、加载骨架、错误、权限不足,而非只画理想态。
- [ ] 间距/圆角/字号成体系且偏紧凑,不是处处 16px/24px 的默认脚手架感。

## 7. 待核对项(诚实标注)

- 精确色值继承自 JumpServer / INSPINIA 主题体系,以实际组件库取色为准,落地时用取色器校准。
- JumpServer 新版(Lina/Vue)与经典版视觉略有差异;本规范取其**经典绿 + 深色侧栏**的稳定特征,不追具体版本像素级还原。

> 来源:JumpServer 官方外观文档确认「经典绿」为默认主题,并提供中国红/深邃黑/科技蓝/经典绿/高贵紫五套主题。
